SBL Admin Bar allows you to dynamically turn the Admin Bar on/off using the keys control-shift-a

== Description ==

Is the Admin Bar in the way while designing the site?  Enter control-shift-a while logged into
the front end of the site to toggle the admin bar on/off.  You will see a temporary dialog confirming
if the admin bar is on or off.  Refresh the page to see the result.

You can enable or disable this feature by user role from the admin under Settings => SBL Admin Bar

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **SBL Admin Bar** in the search box and click on search button.
3. Find SBL Admin Bar plugin.
4. Then click on Install Now and after that activate the plugin.

OR

1. Download and save the **SBL Admin Bar** plugin to your hard disk.
2. Login to your WordPress and go to the Add Plugins page.
3. Click Upload Plugin button to upload the zip.
4. Click Install Now to install and activate the plugin.
